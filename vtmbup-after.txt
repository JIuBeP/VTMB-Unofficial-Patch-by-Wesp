To play the Unofficial Patch version of Bloodlines
START BLOODLINES FROM THE SHORTCUT THE PATCH ADDS!

If you want to play Bloodlines directly from Steam
ADD -game Unofficial_Patch TO IT'S LAUNCH OPTIONS.

In case you are playing Bloodlines from GOG Galaxy
or Steam you need to DISABLE THE GAME AUTO UPDATE.

If you want to play Bloodlines UNDER LINUX USE THE
GAME MOD LOADER tool shortcut from the start menu.

After updating the patch version or switching from
basic to plus YOU ALWAYS NEED TO START A NEW GAME!
